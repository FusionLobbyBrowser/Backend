# Backend

This is the backend for [FLB](https://fusion.hahoos.dev/). It features a lot of features, such as:
- Connecting with [Steamworks](https://github.com/rlabrecque/Steamworks.NET) or [SteamKit](https://github.com/SteamRE/SteamKit)
- An IMAP Client allowing for faster login when using SteamKit
- Error handling
